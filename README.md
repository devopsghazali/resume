# DevOps Ghazali Resume

This is a React + Vite portfolio focused on practical DevOps projects, real repository links, and interview-friendly explanations.

## What it shows

- AWS and DevSecOps delivery flow
- Docker build, scan, and publish pipeline
- Kubernetes observability stack
- Terraform multi-environment AWS baseline
- GitOps and Kubernetes hardening

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

Each project card opens a short explanation and a GitHub repository link. The project notes pages add the longer architecture and deployment story for interview prep.
