# Networking, SSL, and Domain

DNS, routing, TLS, proxies, and traffic basics.

## Overview

This track keeps 3 practical projects and 10 interview questions around Networking, SSL, and Domain.

## Projects

### Project 1: DNS and Reverse Proxy Map

Map a domain to an application through Nginx reverse proxy routing.

- Stack: DNS, Nginx, Reverse Proxy, HTTP
- Resume link: https://github.com/devopsghazali/resume/blob/main/labs/networking-ssl-domain/README.md#dns-and-reverse-proxy-map

### Project 2: TLS Certificate Renewal Lab

Issue and renew certificates and verify secure browser access.

- Stack: SSL, TLS, certbot, HTTPS
- Resume link: https://github.com/devopsghazali/resume/blob/main/labs/networking-ssl-domain/README.md#tls-certificate-renewal-lab

### Project 3: Firewall and Port Troubleshooter

Diagnose blocked traffic, wrong ports, and service reachability issues.

- Stack: Networking, firewall, netstat, curl
- Resume link: https://github.com/devopsghazali/resume/blob/main/labs/networking-ssl-domain/README.md#firewall-and-port-troubleshooter
