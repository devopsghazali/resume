# Multi-tenant SaaS Subscription Engine

Node.js backend for tenant-aware billing, subscriptions, and access control.

## Project

This professional project focuses on tenancy boundaries, auth, billing workflows, and observability.

## Stack

- Node.js
- Express
- PostgreSQL
- RBAC
- Observability
